## Descripcion
Segunda prueba practica que incluye conceptos avanzados de React como manejo de formularios, eventos y comunicacion entre componentes.

## Caracteristicas Implementadas
- Formularios controlados
- Validacion de inputs
- Manejo de eventos
- Listas y keys
- Conditional rendering

## Como Ejecutar
1. Clonar repositorio
2. `npm install`
3. `npm start`
4. Abrir http://localhost:3000

## Scripts Disponibles
- `npm start` - Servidor de desarrollo
- `npm run build` - Build de produccion
- `npm test` - Ejecutar pruebas

## Dependencias
- React
- React DOM
- React Scripts

## Criterios de Evaluacion
- Correcto manejo de formularios
- Validacion de datos
- Estructura de componentes
- Calidad del codigo
